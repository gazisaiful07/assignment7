# 👥 KeenKeeper — Keep Your Friendships Alive

A modern and responsive friendship management web application built with React.js.  
KeenKeeper helps users maintain meaningful relationships by tracking communication history, setting relationship goals, and managing friend interactions through calls, texts, and video check-ins.

## ✨ Features

- 👥 View and manage friend profiles
- ⚡ Quick Call, Text, and Video check-ins
- 📜 Timeline tracking for all interactions
- 📊 Friendship analytics with Pie Charts
- 🔍 Filter timeline interactions by type
- 📱 Fully responsive design for mobile, tablet, and desktop
- 🔔 Toast notifications for interactions
- ❌ Custom 404 error page
- ⏳ Loading animation while fetching data
